Recargas Fenie Energia

Sistema de Control Centralizado de una red de Puntos de Recarga de Coches Eléctricos. Backend realizado en Spring utilizando JSON y SOAP para la comunicación con los puntos. Frontend desarrollado en Laravel.

Permite la gestión total de los puntos de recarga asociados a la red utilizando el protocolo OCPP (Open Charge Point Protocol). Los clientes pueden realizar la búsqueda de puntos cercanos a su ubicación, obtener toda la información sobre estos, reservar o realizar recargas. Permite además la interoperabilidad con otros CPOs (Charge Point Operator) utilizando el intercambio de información mediante la especificiación OCPI (Open Charge Point Interface).

Se realizó además el desarrollo de una API Rest en Spring para ser utilizada por las aplicaciones móbiles

Aplicación premiada con "Desarrollo de negocio en Internet de las Cosas" en la IV edición de los premios Data Center Market

  • Tecnologías: Java Spring, Laravel, Redis, MySQL, Google Maps, Google Cloud Messaging, SOAP, Websocket
  • Disponible en: https://recarga.fenieenergia.es

SATIC

Sistema de Análisis de Trazas de Control de Copextel (SATIC). Aplicación para el procesamiento y consulta de los logs de acceso del proxy Squid. Permite realizar consultas a los datos importados, posibilitando mantener un monitoreo y control sobre el uso del servicio de Internet en la corporación Copextel S.A.

SATIC permite además interactuar con Squid, posibilitando la creación de usuarios, eliminación, desactivación de cuentas y la gestión de las listas de control de acceso (ACLs) relativas a la IPS autorizadas a acceder a los proxys

Contiene funcionalidades agregadas como la exportación de los datos en diferentes formatos, generación de reportes de control y análisis, sistema de alertas, canal RSS, entre otras.

Adicionalmente se programó además un complemento que puede ser instalado en los navegadores Chrome, Firefox y en la nueva versión de Microsoft Edge y que posibilita a los usuarios consultar sus datos principales, así como chequear el consumo cada cierto intervalo de tiempo configurado.

Similar a esta extensión se desarrolló una aplicación de escritorio en Java para la consulta y chequeo del consumo de la cuota de Internet.

  • Tecnologías: Laravel, MongoDB, JavaScript, Java

PingWheel

Red social que permite encontrar e interactuar con personas con intereses en común. Al registrarse en la red, el usuario debe suministrar información acerca de su ubicación y datos personales clasificados en 6 categorías (Work, School, Personal, Interest, Background and Hobby). La aplicación es capaz de mostrar coincidencias de otros usuarios que compartan cosas en común, basada en un sistema de puntuación asignado a los valores de cada categoría y a un algoritmo programado para determinar el porciento de compatibilidad.

También se desarrollo una API Rest utilizando Swagger y OpenAPI Specification

  • Tecnologías: Laravel, JavaScript, Mysql Functions, Mysql Store Procedures, Mysql Triggers
  • Disponible en: http://pingwheel.com
